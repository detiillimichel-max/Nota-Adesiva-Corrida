# Nota Adesiva Corrida

Jogo de arrastar post-its coloridos para o quadro antes do tempo acabar. Cada cor vale pontos diferentes e cada fase tem metas específicas.

## Como Jogar

1. Arraste um post-it da paleta para o quadro de cortiça
2. Solte dentro do quadro para colar
3. Complete a meta de cada cor antes do tempo acabar
4. Avance de fase. As metas aumentam a cada fase

## Pontuação

- **Amarelo**: 10 pts
- **Rosa**: 20 pts
- **Azul**: 30 pts
- **Verde**: 50 pts

## Metas por Fase

- **Fase 1**: Rosa 3, Azul 3
- **Fase 2**: Rosa 5, Azul 5, Amarelo 5
- **Fase 3**: Rosa 8, Azul 8, Amarelo 5, Verde 3
- **Fase 4+**: Metas aumentam progressivamente

## Regras

- Tempo por fase: 45 segundos
- Cumpriu todas as metas → Próxima fase
- Tempo esgotou → Game Over
- Pontuação e fase são exibidas no HUD

## Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Drag & Drop com Pointer Events
- 100% offline, sem CDN externo
- Responsivo para mobile e desktop

## Instalação

É um único arquivo `index.html`. Basta abrir no navegador ou hospedar no GitHub Pages.
